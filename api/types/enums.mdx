## Overview

This page documents all enumeration types defined in the Wire library. These enums provide type-safe constants for various entity properties, connection categories, and workflow options.

All enums are defined in `wire/core/entities.hpp` (lines 14-143).

---

## ConnectionCategory

**Type:** `std::uint8_t`  
**Definition:** `entities.hpp:14-20`

Classifies the electrical or communication category of a connection.

| Value | Description |
|-------|-------------|
| `kHighVoltage` (0) | High voltage power distribution lines |
| `kLowVoltage` (1) | Low voltage power distribution or service drops |
| `kCommunication` (2) | Telephone, cable, or other communication lines |
| `kOptical` (3) | Fiber optic communication cables |
| `kDrop` (4) | Service drop connections to buildings |
