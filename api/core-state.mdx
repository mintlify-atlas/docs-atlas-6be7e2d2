## Overview

`CoreState` is the primary mutable state container for managing utility network topology. It provides methods for creating, editing, and querying poles, ports, anchors, spans, bundles, and attachments. All mutations return `EditResult<T>` to track changes and propagate errors.

**Location:** `wire/core/core_state.hpp:223-374`

**Namespace:** `wire::core`

## Constructor

```cpp
CoreState()
```

Creates a new empty core state with default settings and registered pole/bundle templates.

## Entity Creation Methods

### AddPole

```cpp
EditResult<ObjectId> AddPole(
  const Transformd& world_transform,
  double height_m = 10.0,
  std::string_view name = {},
  PoleKind kind = PoleKind::kGeneric,
  PlacementMode placement_mode = PlacementMode::kAuto
)
```

Creates a new pole in the network.

<ParamField path="world_transform" type="const Transformd&" required>
  World-space position, rotation, and scale
</ParamField>

<ParamField path="height_m" type="double" default="10.0">
  Height of the pole in meters
</ParamField>

<ParamField path="name" type="std::string_view" default="">
  Optional display name for the pole
</ParamField>

<ParamField path="kind" type="PoleKind" default="PoleKind::kGeneric">
  Type of pole (kGeneric, kWood, kConcrete, kSteel)
</ParamField>

<ParamField path="placement_mode" type="PlacementMode" default="PlacementMode::kAuto">
  Placement mode (kAuto for ground snapping, kManual for exact positioning)
</ParamField>

<ResponseField name="EditResult<ObjectId>" type="EditResult<ObjectId>">
  Result containing the new pole's ObjectId on success, or error details on failure
</ResponseField>

**Example:**

```cpp
auto result = state.AddPole(
  Transformd{{10.0, 20.0, 0.0}},
  12.0,
  "Pole_001",
  PoleKind::kWood,
  PlacementMode::kAuto
);
if (result.ok) {
  ObjectId pole_id = result.value;
  // Use pole_id...
}
```

## Usage Example

```cpp
#include "wire/core/core_state.hpp"

using namespace wire::core;

CoreState state;

// Create two poles
auto pole_a = state.AddPole(Transformd{{0, 0, 0}}, 12.0);
auto pole_b = state.AddPole(Transformd{{30, 0, 0}}, 12.0);

if (!pole_a.ok || !pole_b.ok) {
  // Handle errors
  return;
}

// Connect poles with a span
auto conn = state.AddConnectionByPole(
  pole_a.value,
  pole_b.value,
  ConnectionCategory::kLowVoltage
);

if (!conn.ok) {
  std::cerr << "Connection failed: " << conn.error << std::endl;
  return;
}

// Commit changes and run validation
CommitOptions commit_opts;
commit_opts.run_recalc = true;
commit_opts.run_validate_fast = true;

auto result = state.Commit(commit_opts);

if (!result.validation.ok()) {
  for (const auto& issue : result.validation.issues) {
    std::cerr << "Validation: " << issue.message << std::endl;
  }
}

std::cout << "Recalculated " << result.recalc_stats.total_processed() 
          << " items" << std::endl;
```
